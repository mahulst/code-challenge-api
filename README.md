# API for code challenge

Small api that serves randomized fake flight information and an endpoint to search for gate changes.

# Install

`$ npm install`

# Start

`$ npm run start`

# Routes

-   `/arrivals`
-   `/departures`
-   `/gate-changes`
-   `/gate-changes/:search`
